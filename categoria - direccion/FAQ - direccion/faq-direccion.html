Preguntas fecuentes

¿Qué es la dirección de un auto?

Es el conjunto de sistemas que permiten girar las ruedas directrices de un vehículo. Gracias a los sistemas de dirección, el vehículo puede girar

¿Qué elementos componen la dirección del auto?

Los elementos principales son el volante, caja de dirección, brazo de mando, barra de acoplamiento, rótulas, cremallera, una columna de dirección y rótulas.

¿Qué tipos de sistema de dirección para el auto existen?

Los sistemas de dirección pueden ser mecánica, hidráulica y eléctrica.

¿Qué repuestos necesito para reparar la transmisión manual de un auto?

Los repuestos más comunes que se necesitan para reparar la transmisión manual de un carro suelen ser: piola selectora, rodamientos, ejes, retenes, soporte para caja de cambios, piñones de caja, entre otros.

¿Qué repuestos necesito para reparar la transmisión o caja automática de un auto?

Para reparar o realizarle mantenimiento a una caja automática puedes necesitar, la carcaza de transmisión, correas, retenes y líquidos o aceite de caja de cambios automática.
