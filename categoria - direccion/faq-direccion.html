Preguntas frecuentes

¿Qué es la dirección de un auto?

Es el conjunto de sistemas que permiten girar las ruedas de un vehículo. Gracias a los sistemas de dirección, el vehículo puede girar.

¿Qué elementos componen la dirección del auto?

Los elementos principales son: el volante, caja de dirección, brazo de mando, barra de acoplamiento, rótulas, cremallera, una columna de dirección y rótulas.

¿Qué tipos de sistema de dirección para el auto existen?

Los sistemas de dirección pueden ser mecánica, hidráulica y eléctrica.
