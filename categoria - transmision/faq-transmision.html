Preguntas frecuentes

¿Qué es la transmisión de un auto?

La transmisión es el sistema encargado de llevar el movimiento a las ruedas del vehículo, transmitiendo la potencia y fuerza generada por el motor. Es responsable de permitir que el auto se mueva o se desplace.

¿Qué elementos componen la transmisión del auto?

Los elementos que componen el sistema de transmisión de un vehículo son: el embrague, caja de cambio, diferencial, árbol de transmisión y palieres.

¿Qué tipos de sistemas de transmisión para el auto existen?

Existen dos tipos principales de sistemas de transmisión para autos. El sistema de transmisión manual, donde el conductor modifica la relación de transmisión utilizando una palanca de cambios y el embrague. Y el sistema de transmisión automático, donde los cambios de marcha se realizan de forma automática sin la intervención del conductor y no se requiere pedal de embrague.

¿Qué repuestos necesito para reparar la transmisión manual de un auto?

Los repuestos más comunes que se necesitan para reparar la transmisión manual de un carro suelen ser: piola selectora, rodamientos, ejes, retenes, soporte para caja de cambios, piñones de caja, entre otros.

¿Qué repuestos necesito para reparar la transmisión o caja automática de un auto?

Para reparar o realizar mantenimiento a una caja de cambios automática, puedes necesitar repuestos como la carcasa de transmisión, correas, retenes y líquidos o aceite de la caja de cambios automática.

¿Qué es el kit de embrague?

El kit de embrague es el conjunto de componentes que se encarga de transmitir la potencia del motor a la caja de cambios, y a su vez, a las ruedas del vehículo.

¿Qué incluye el kit de embrague?

El kit de embrague incluye el disco de embrague, cojinete y la placa de presión.
