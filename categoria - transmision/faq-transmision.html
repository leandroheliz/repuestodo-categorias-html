Preguntas frecuentes

¿Qué es la transmisión de un auto?

Es el sistema encargado de llevar el movimiento a las ruedas del vehículo, transmitiendo la potencia y fuerza que genera el motor. De esta manera logra que el auto se mueva o desplace.

¿Qué elementos componen la transmisión del auto?

Los elementos que componen el sistema de transmisión de un vehículo son: el embrague, caja de cambio, diferencial, árbol de transmisión y palieres.

¿Qué tipos de sistema de transmisión para el auto existen?

Podemos encontrarnos con el sistema de transmisión manual, donde el conductor modifica la relación de transmisión por medio de una palanca de cambios y el embrague. Por otro lado tenemos el sistema de transmisión automático. Aquí no existe el pedal de embrague y los cambios de marcha se realizan por sí solos.

¿Qué repuestos necesito para reparar la transmisión manual de un auto?

Los repuestos más comunes que se necesitan para reparar la transmisión manual de un carro suelen ser: piola selectora, rodamientos, ejes, retenes, soporte para caja de cambios, piñones de caja, entre otros.

¿Qué repuestos necesito para reparar la transmisión o caja automática de un auto?

Para reparar o realizarle mantenimiento a una caja automática puedes necesitar: la carcaza de transmisión, correas, retenes y líquidos o aceite de caja de cambios automática.

¿Qué es el kit de embrague?

El kit de embrague se encarga de transmitir la potencia del motor a la caja de cambios, está a su vez a la transmisión y a las ruedas.

¿Qué incluye el kit de embrague?

El kit de embrague incluye el disco de embrague, cojinete y el plato de maza o presión.
