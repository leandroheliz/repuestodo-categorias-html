Preguntas fecuentes

¿Qué es la transmisión de un auto?

Es el sistema encargado de llevar el movimiento a las ruedas del vehículo, transmitiendo la potencia y fuerza que genera el motor y así lograr que el auto se mueva o desplaze.

¿Que elementos componen la transmisión del auto?

Los elementos que componen el sistema de transmisión de un vehiculo son: el embrague, caja de cambio, diferencial, arbol de transmisión y palieres.

¿Que tipos de sistema de transmisión para el auto existen?

Podemos encontrarnos con el sistema de transmisión manual, donde el conductor modifica la relación de transmisión por medio de una palanca de cambios y el embrague. Por otro lado tenemos el sistema de transmisión automatico, aquí no existe el pedal de embrague y los de marcha se realizan por sí solos.

¿Que repuestos necesito para reparar la transmisión manual de un auto?

Los repuestos más comunes que se necesitan para reparar la transmisión manual de un carro suelen ser: piola selectora, rodamientos, ejes, retenes, soporte para caja de cambios, piñones de caja, entre otros.

¿Que repuestos necesito para reparar la transmisión o caja automatica de un auto?

Para reparar o realizarle mantenimiento a una caja automatica puedes necesitar, la carcaza de transmisión, correas, retenes y liquidos o aceite de caja de cambios automática.

¿Que es el kit de embrague?

El kit de embrague se encarga de transmitir la potencia del motor a la caja de cambios, esta a su vez a la transmisión y a las ruedas.

¿Que incluye el kit de embrague?

El kit de embrague incluye el disco de embrague, cojienete y el plato de maza o presión.
